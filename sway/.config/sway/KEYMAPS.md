# Sway Keybindings Cheat Sheet

> **Mod** = Super (Win key)

## Basic Operations

| Keybind | Action |
|---------|--------|
| `Mod + Tab` | Next workspace |
| `Alt + Tab` | Previous workspace |
| `Mod + Shift + Q` | Kill focused window |
| `Mod + Shift + R` | Reload config |

## Applications

| Keybind | Action |
|---------|--------|
| `Mod + Enter` | Terminal (Alacritty) |
| `Mod + W` | Browser (Firefox) |
| `Mod + N` | File Manager (Thunar) |
| `Mod + Shift + B` | Bluetooth Manager |
| `Mod + Shift + N` | Notification center |
| `Mod + Shift + E` | Power menu |
| `Mod + Shift + X` | Lock screen |

## Launcher (Rofi)

| Keybind | Action |
|---------|--------|
| `Mod + D` | App launcher (drun) |
| `Alt + D` | Run launcher |
| `Alt + W` | Window switcher |

## Focus

| Keybind | Action |
|---------|--------|
| `Mod + H / ←` | Focus left |
| `Mod + J / ↓` | Focus down |
| `Mod + K / ↑` | Focus up |
| `Mod + L / →` | Focus right |
| `Mod + Q` | Toggle focus tiling/floating |
| `Mod + U` | Focus parent container |
| `Mod + I` | Focus child container |

## Move Window

| Keybind | Action |
|---------|--------|
| `Mod + Shift + H / ←` | Move window left |
| `Mod + Shift + J / ↓` | Move window down |
| `Mod + Shift + K / ↑` | Move window up |
| `Mod + Shift + L / →` | Move window right |

## Resize Window

| Keybind | Action |
|---------|--------|
| `Mod + Ctrl + H / ←` | Grow width |
| `Mod + Ctrl + L / →` | Shrink width |
| `Mod + Ctrl + K / ↑` | Grow height |
| `Mod + Ctrl + J / ↓` | Shrink height |

## Layout

| Keybind | Action |
|---------|--------|
| `Mod + Z` | Split horizontal |
| `Mod + A` | Split vertical |
| `Mod + M` | Toggle fullscreen |
| `Mod + E` | Tabbed layout |
| `Mod + T` | Toggle split layout |
| `Mod + Space` | Toggle tiling/floating (floats at 1300x800, centered) |

## Workspaces

| Keybind | Action |
|---------|--------|
| `Mod + 1–8` | Switch to workspace 1–8 |
| `Mod + Shift + 1–8` | Move window to workspace 1–8 |

## Screenshots

| Keybind | Action |
|---------|--------|
| `Print` | Screenshot to file |
| `Mod + Print` | Screenshot selection to file |
| `Shift + Print` | Screenshot to clipboard |
| `Mod + Shift + Print` | Screenshot selection to clipboard |

## Function Keys

| Keybind | Action |
|---------|--------|
| `Brightness Up/Down` | Adjust screen brightness ±5% |
| `Volume Up/Down` | Adjust volume ±5% |
| `Mute` | Toggle mute |
| `Play/Pause` | Toggle media playback |
| `Next / Prev` | Next/previous track |
