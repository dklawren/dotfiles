Sway Keybindings Cheat Sheet

Mod = Super (Win key)

Basic Operations

Mod + TabNext workspace
Alt + TabPrevious workspace
Mod + Shift + QKill focused window
Mod + Shift + RReload config

Applications

Mod + EnterTerminal (Alacritty)
Mod + WBrowser (Firefox)
Mod + NFile Manager (Thunar)
Mod + Shift + BBluetooth Manager
Mod + Shift + NNotification center
Mod + Shift + EPower menu
Mod + Shift + XLock screen

Launcher (Rofi)

Mod + DApp launcher (drun)
Alt + DRun launcher
Alt + WWindow switcher

Workspaces

Mod + 1–8Switch to workspace
Mod + Shift + 1–8Move window to workspace

Focus

Mod + H / ←Focus left
Mod + J / ↓Focus down
Mod + K / ↑Focus up
Mod + L / →Focus right
Mod + QToggle focus tiling/floating
Mod + UFocus parent container
Mod + IFocus child container

Move Window

Mod + Shift + H / ←Move left
Mod + Shift + J / ↓Move down
Mod + Shift + K / ↑Move up
Mod + Shift + L / →Move right

Resize Window

Mod + Ctrl + H / ←Grow width
Mod + Ctrl + L / →Shrink width
Mod + Ctrl + K / ↑Grow height
Mod + Ctrl + J / ↓Shrink height

Layout

Mod + ZSplit horizontal
Mod + ASplit vertical
Mod + MToggle fullscreen
Mod + ETabbed layout
Mod + TToggle split layout
Mod + SpaceToggle tiling/floating

Screenshots

PrintScreenshot to file
Mod + PrintScreenshot selection to file
Shift + PrintScreenshot to clipboard
Mod + Shift + PrintSelection to clipboard

Function Keys

Brightness ↑/↓Brightness ±5%
Volume ↑/↓Volume ±5%
MuteToggle mute
Play/PauseToggle playback
Next / PrevNext/previous track
